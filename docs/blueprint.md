# **App Name**: MarketPulse

## Core Features:

- Market Insight Dashboard: An intuitive, high-density dashboard UI featuring live performance metrics and simplified market movement summaries.
- AI Market Synthesis Tool: A generative AI tool that summarizes voluminous financial reports and news articles into clear, actionable bullet points.
- Intelligent News Stream: A comprehensive UI containing search-enabled news cards, bookmarks, and industry-specific category filters.
- Tiered IPO Explorer: A structured view designed to monitor live, upcoming, and recently listed IPOs with status tracking.
- Sentiment Pulse Tool: An AI-powered reasoning tool that analyzes the underlying tone of news trends to provide an objective market sentiment score.
- Client-Side Bookmarking: User interface elements designed to capture and display saved news items for personalized reading lists.
- Cross-Platform Adaptive Interface: A fully responsive mobile and desktop layout inspired by high-end financial tools, ensuring accessibility on all devices.

## Style Guidelines:

- Primary color: Growth Emerald (#18cf9c), a vibrant and professional green representing financial upward momentum.
- Background color: Dark Pine (#131c19), a deeply desaturated version of the primary hue used to establish a sophisticated dark-themed workspace.
- Accent color: Neon Mint (#4dd44d), an analogous, highly saturated shade designed to highlight call-to-action buttons and positive price shifts.
- Font pairing: 'Space Grotesk' (sans-serif) for high-impact, technical-looking headlines, and 'Inter' (sans-serif) for neutral, highly legible body text and numerical data.
- Sleek, feather-light stroke icons that maintain visual clarity at small sizes on mobile dashboards.
- A balanced masonry grid for the news hub and a flexible sidebar navigation system for the dashboard and IPO sections.
- Smooth height transitions and opacity fades for filtering categories to ensure a fluid user experience.
