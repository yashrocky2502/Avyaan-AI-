# MarketPulse Pro Terminal

MarketPulse is a high-performance financial intelligence terminal built with NextJS and ShadCN UI.

## Features

- **Command Center Dashboard**: Real-time market snapshots and intelligence.
- **Intelligent News Stream**: Synthesized news with sentiment and impact analysis.
- **Market Scanner**: Professional screening for gainers, losers, and corporate actions.
- **Research Hub**: Institutional-grade stock and sector benchmarking.
- **Daily Briefing**: Curated morning notes for active investors.

## Technical Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI / ShadCN
- **Icons**: Lucide React
- **Data Layer**: Service-oriented architecture with Mock Repositories.
